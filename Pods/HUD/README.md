# Installation

```
pod 'HUD', '~> 1.0.0'
```

# Using

```
// show
HUD.show(.loading, text: "Loading...")
HUD.show(.success, text: "Success")
HUD.show(.none, text: "Text...")

// timed dismiss
HUD.show(.loading, text: "Loading...", time: 3) { _ in
            print("Loading completed")
        }


// dismiss
HUD.dismiss()
```
